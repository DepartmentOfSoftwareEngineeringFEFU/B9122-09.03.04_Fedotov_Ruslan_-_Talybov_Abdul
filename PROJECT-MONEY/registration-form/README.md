# Frontend

React-интерфейс проекта Trade.

## Запуск

```powershell
cd C:\Учеба\trade\PROJECT-MONEY\registration-form
npm install
npm start
```

Frontend будет доступен по адресу:

```text
http://127.0.0.1:3000
```

Backend должен быть запущен отдельно на:

```text
http://127.0.0.1:8000
```

## Команды

```powershell
npm start
```

Запускает dev-сервер.

```powershell
npm run build
```

Собирает production-версию в папку `build`.

```powershell
npm test
```

Запускает тесты Create React App.
